## Personal Knowledge System (PKS)

A PKS is a digital knowledge repository used to organize and connect learning materials.

Requirements:
- Markdown format
- Self-contained folder
- Compatible with Obsidian
- Includes HOME.md or START HERE.md

## Typical Class Flow

1. Recall and Recite
2. Lecture and Theory
3. Hands-On Lab
4. Reflection and PKS
5. Assessment and Q&A
